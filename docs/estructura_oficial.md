# Estructura oficial del proyecto

## Estructura que se usará en adelante

La estructura oficial del sistema será:

- backend/core
- backend/apps
- backend/templates
- backend/static
- backend/media
- backend/manage.py

## Estructura anterior
La carpeta backend/modules se conserva solo como referencia histórica inicial y no será la base de desarrollo futuro.

## Regla de trabajo
Todo nuevo desarrollo debe realizarse dentro de backend/apps y backend/core.
