# Usuarios y roles del sistema

## Objetivo
Permitir acceso controlado al sistema mediante usuario y contraseña.

## Roles iniciales

### Administrador
Puede gestionar usuarios, catálogos y proyectos.

### Formulador
Puede crear y editar proyectos.

### Revisor
Puede revisar proyectos y emitir observaciones.

### Consulta
Puede visualizar información sin editar.

## Relación con proyectos
Cada proyecto podrá registrar qué usuario lo creó.

## Futuro esperado
- Inicio de sesión real
- Control de permisos por rol
- Historial de acciones por usuario
- Recuperación de contraseña
