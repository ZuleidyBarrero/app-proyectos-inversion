# Prueba nocturna de archivado lógico

## Objetivo
Verificar que un proyecto pueda pasar a estado Archivado sin eliminarse.

## Casos a probar

### Caso 1
Abrir el detalle de un proyecto activo.
Resultado esperado: ver enlace para archivar.

### Caso 2
Entrar a la confirmación de archivado.
Resultado esperado: ver mensaje explicando la acción.

### Caso 3
Confirmar archivado.
Resultado esperado: el proyecto cambia a estado Archivado.

### Caso 4
Buscar el proyecto en el listado.
Resultado esperado: sigue existiendo y puede filtrarse por estado Archivado.
