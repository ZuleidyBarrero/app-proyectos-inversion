# Estructura inicial de anexos del proyecto

## Objetivo
Permitir que cada proyecto tenga documentos y soportes asociados.

## Tipos de anexos esperados
- PDF
- Excel
- Imágenes
- Actas
- Certificados
- Estudios previos
- Otros documentos técnicos

## Futuro esperado
- Almacenamiento en nube
- Asociación de múltiples archivos por proyecto
- Registro de nombre del archivo
- Registro de fecha de carga
- Registro del usuario que cargó el archivo
