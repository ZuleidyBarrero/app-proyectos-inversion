# Prueba nocturna de anexos de proyecto

## Pasos
1. Entrar a /projects/
2. Abrir un proyecto existente o crear uno nuevo
3. Entrar al detalle del proyecto
4. En la sección de anexos, seleccionar un archivo
5. Escribir una descripción
6. Subir el anexo

## Verificar
- que el anexo quede listado en el detalle del proyecto
- que el enlace abra el archivo
- que en admin aparezca asociado al proyecto
