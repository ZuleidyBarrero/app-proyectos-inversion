# Hoja de ruta del sistema de proyectos de inversión pública

## Fase 1 - Estructura
- Crear repositorio
- Crear ramas
- Organizar carpetas
- Definir arquitectura
- Crear apps conceptuales
- Preparar estructura tipo Django

## Fase 2 - Ejecución real
- Llevar el proyecto a un entorno con terminal
- Instalar Django y dependencias
- Crear proyecto Django real
- Ajustar settings.py
- Configurar base de datos
- Probar servidor local en nube

## Fase 3 - Funcionalidad real
- Modelo Project real
- Modelo User real o integración con auth de Django
- Formularios reales
- URLs reales
- Templates conectados a vistas
- Login funcional
- CRUD de proyectos

## Fase 4 - Funcionalidad institucional
- Catálogos geográficos reales
- Carga de anexos
- Roles y permisos
- Historial de estados
- Reportes
- Exportaciones

## Fase 5 - Despliegue
- Base de datos en nube
- Almacenamiento de archivos
- Dominio o subdominio
- Usuarios institucionales
- Ambiente productivo
