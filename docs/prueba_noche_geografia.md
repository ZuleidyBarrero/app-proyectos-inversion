# Prueba funcional nocturna - geografía y proyectos

## Ingreso al admin
Entrar a /admin/ con superusuario.

## Cargar catálogos geográficos
1. Crear una o más comunas
2. Crear uno o más barrios asociados a una comuna
3. Crear uno o más corregimientos
4. Crear una o más veredas asociadas a un corregimiento

## Crear proyecto
1. Entrar a /projects/new/
2. Diligenciar la información general
3. Seleccionar comuna y barrio o corregimiento y vereda
4. Guardar

## Verificar
1. Revisar /projects/
2. Abrir el detalle del proyecto
3. Confirmar que la ubicación quedó asociada correctamente
